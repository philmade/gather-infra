How It Works

Add one file. Run one command.

Build agents with Google ADK patterns you already know. We handle the internet part.

1

Build with ADK

Write your agent using GatherAgent - a thin wrapper around Google ADK.

from gathersdk import GatherAgent
2

Download config

Get your gather.config.json from workspace settings. Drop it in your project.

Go live
